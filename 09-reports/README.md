# Отчеты и Summaries

**Версия:** 1.0.0  
**Дата создания:** 2025-11-06

---

## Описание

Централизованное хранение всех отчетов о работе над проектом NECPGAME.

---

## Структура

### `session-reports/`
Отчеты по рабочим сессиям, сгруппированные по датам.

**Формат:** `YYYY-MM-DD/` - Директория для отчетов дня

**Содержит:**
- Session summaries (краткие, расширенные, полные)
- Финальные отчеты по завершению работы
- Отчеты о готовности документов

### `next-steps/`
Планы и следующие шаги проекта.

**Содержит:**
- Текущие планы действий
- Приоритетные задачи
- Стратегические направления

---

## Использование

### Создание нового отчета о сессии

1. Создать директорию для даты: `session-reports/YYYY-MM-DD/`
2. Разместить отчеты в директории
3. Создать `README.md` с описанием файлов

### Обновление планов

Обновить файл `next-steps/NEXT-STEPS.md` с актуальными планами.
