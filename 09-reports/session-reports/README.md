# Session Reports

**Версия:** 1.0.0  
**Дата создания:** 2025-11-06

---

## Описание

Отчеты по рабочим сессиям, сгруппированные по датам.

---

## Структура

Каждая директория соответствует дате работы:

```
session-reports/
├── 2025-11-06/           # Отчеты за 6 ноября 2025
│   ├── SESSION-SUMMARY-2025-11-06.md
│   ├── MEGA-SESSION-SUMMARY-2025-11-06.md
│   ├── ULTIMATE-SESSION-SUMMARY-2025-11-06.md
│   ├── FINAL-SESSION-REPORT-2025-11-06.md
│   ├── BRAIN-READINESS-FINAL-REPORT-2025-11-06.md
│   └── README.md
└── README.md             # Этот файл
```

---

## Формат названий

- **Директории:** `YYYY-MM-DD/`
- **Файлы:** `[TYPE]-[DATE].md`

**Типы файлов:**
- `SESSION-SUMMARY` - Краткая сводка сессии
- `MEGA-SESSION-SUMMARY` - Расширенная сводка
- `ULTIMATE-SESSION-SUMMARY` - Полная детальная сводка
- `FINAL-SESSION-REPORT` - Финальный отчет по завершению
- `BRAIN-READINESS-FINAL-REPORT` - Отчет о готовности документов к API

---

## Навигация

Для быстрого поиска отчетов используйте формат даты `YYYY-MM-DD`.

**Пример:** Отчеты за 6 ноября 2025 → `2025-11-06/`
