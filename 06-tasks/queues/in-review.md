# Очередь документов — статус `in-review`

- Лимит файла: ≤ 500 строк. При превышении создайте `in-review_0001.md`, `in-review_0002.md`.
- Формат записи:

```markdown
- **Документ:** .BRAIN/02-gameplay/social/mentorship-system-детально.md (v1.0.0)  
  **Отправлено на ревью:** 2025-11-09 00:10 — Brain Manager  
  **Ответственный за проверку:** Brain Readiness Checker.
```

- После завершения проверки перенесите запись в файл нового статуса и обновите дату.
