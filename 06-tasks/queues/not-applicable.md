# Очередь документов — статус `not-applicable`

- Лимит файла: ≤ 500 строк. При превышении создайте `not-applicable_0001.md`, `not-applicable_0002.md`.
- Формат записи:

```markdown
- **Документ:** .BRAIN/06-tasks/README.md (v1.0.0)  
  **Пометка:** 2025-11-09 00:06 — Brain Manager  
  **Причина:** Служебный файл, не предназначен для постановки API задач.
```

- При изменении предназначения документа перенесите запись в актуальный файл статуса.

- **Документ:** .BRAIN/02-gameplay/combat/combat-abilities/README.md (v1.0.1)  
  **Пометка:** 2025-11-09 03:09 — Brain Manager  
  **Причина:** Навигационный индекс каталога боевых способностей, документ не формирует API задачи.
- **Документ:** .BRAIN/02-gameplay/combat/combat-ai-enemies/README.md (v1.0.1)  
  **Пометка:** 2025-11-09 03:09 — Brain Manager  
  **Причина:** Индекс материалов по AI врагам; служебный документ, API задачи не требуются.
- **Документ:** .BRAIN/02-gameplay/economy/ECONOMY-EXPANSION-SUMMARY.md (v2.0.0)  
  **Пометка:** 2025-11-09 03:27 — Brain Manager  
  **Причина:** Сводка расширения экономики агрегирует прогресс раздела и не формирует самостоятельных API задач.
