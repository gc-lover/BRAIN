# Очередь документов — статус `blocked`

- Лимит файла: ≤ 500 строк. При превышении создайте `blocked_0001.md`, `blocked_0002.md`.
- Формат записи:

```markdown
- **Документ:** .BRAIN/05-technical/backend/mail-system.md (v1.0.0)  
  **Проверено:** 2025-11-09 00:05 — Brain Manager  
  **Блокер:** Ожидаем финализацию event-схем от narrative команды.
```

- После снятия блокера удалите запись отсюда и перенесите в соответствующий файл нового статуса.
