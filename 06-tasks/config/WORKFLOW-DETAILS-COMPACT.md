## Воркфлоу проекта

```
ИДЕЯ → МЕНЕДЖЕР (.BRAIN) → ЧЕКБРЕЙН (проверка) → 
→ ДУАПИТАСК (создание задач) → АПИТАСК (OpenAPI) → 
→ БЭКТАСК + ФРОНТТАСК (реализация) → КЛИР (архивация)
```

---

## Этапы

1. **МЕНЕДЖЕР:** Создаёт документы, статус draft → approved, api-readiness
2. **ЧЕКБРЕЙН:** Проверяет готовность (опционально)
3. **ДУАПИТАСК:** Создаёт задания для API
4. **АПИТАСК:** Создаёт OpenAPI specs
5. **БЭКТАСК/ФРОНТТАСК:** Реализуют API
6. **КЛИР:** Архивирует завершённые фичи

---

## Статусы

**Документы:** draft → review → approved  
**API Readiness:** needs-work → in-review → ready  
**Implementation:** not-started → in-progress → completed
