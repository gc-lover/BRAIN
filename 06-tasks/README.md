## Структура

### Active (Активные задания)

### Completed (Выполненные задания)
- Архив завершенных задач

### Templates (Шаблоны заданий)

### Config (Конфигурация)
- `ЧЕКБРЕЙН.MD` - **ГЛАВНЫЙ ЧЕКЛИСТ** для агента проверки готовности документов (прикреплять агенту для запуска проверки)
- `readiness-check-guide.md` - Подробная инструкция проверки готовности документов .BRAIN к созданию API задач
- `check-documents-readiness.md` - Краткая инструкция для агента (файл для прикрепления к сообщению)
- `readiness-tracker.yaml` - Трекер готовности документов к созданию API задач

**Система проверки готовности:**
- Используется для определения готовности документов в .BRAIN к созданию API задач для API-SWAGGER
- Документы проверяются по критериям готовности и получают статус (ready, needs-work, blocked, in-review, not-applicable)
- Информация о готовности отслеживается в `readiness-tracker.yaml`
- Готовые документы (status: ready) могут быть использованы для создания API задач
