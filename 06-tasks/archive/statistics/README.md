## Назначение

Эта директория содержит **статистику по всем завершенным работам** проекта NECPGAME.

**Типы статистики:**
1. **Помесячная** - детальная статистика за каждый месяц
2. **Общая** - сводная статистика за все время

## Что отслеживается

### В помесячной статистике:

- Количество завершенных фич за месяц
- Среднее время разработки (от идеи до реализации)
- Среднее время Backend (от начала до завершения)
- Среднее время Frontend (от начала до завершения)
- Количество созданных endpoints
- Количество созданных моделей данных
- Количество миграций БД
- Распределение по категориям (Gameplay, Social, Auth, Economy, и т.д.)
- Топ-5 самых быстрых фич
- Топ-5 самых долгих фич

### В общей статистике:

- Всего фич за все время
- Средние показатели за все время
- Тренды (ускорение/замедление разработки)
- Самые продуктивные месяцы
- Общее количество endpoints/моделей/миграций
