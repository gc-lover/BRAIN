## Контекст

Документ `combat-implants-types.md` содержит только базовое описание типов имплантов:
- Боевые импланты (точность, урон, боевые способности, моды на оружие)
- Тактические импланты (сканирование, хакерство, стелс)
- Защитные импланты (броня, восстановление, иммунитеты)
- Двигательные импланты (скорость, прыжки, паркур, уклонение)
- Операционные системы (OS) (боевые/хакерские/универсальные ОС)

**Требуется:** Детализировать перечень конкретных типов имплантов и их применение для создания API.

**Источники вдохновения:**
- Cyberpunk 2077 (игра) - кибервар и импланты
- Настольная игра Cyberpunk - типы имплантов
- Edgerunners (аниме) - визуальные примеры имплантов

**Интеграция:**
- `combat-implants-limits.md` - ограничения и слоты по типам
- `combat-cyberpsychosis.md` - влияние на человечность
- `combat-abilities.md` - связь с системой способностей
- `equipment-matrix.md` - бренды и сигнатуры имплантов

## Вопрос 1: Детализация боевых имплантов

**Контекст:** Боевые импланты влияют на точность, урон, боевые способности и моды на оружие. Нужно определить конкретные типы имплантов и их применение.

**Варианты:**

**Вариант А: Базовая детализация (3-5 типов на категорию)**
- Простые импланты по категориям (например, для боевых: имплант точности, имплант урона, имплант скорострельности)
- Легко балансировать и реализовывать
- Ограниченное разнообразие
- *Пример:* Минимальная система, как в ранних играх

**Вариант Б: Средняя детализация (5-10 типов на категорию)**
- Больше разнообразия имплантов
- Разные типы для разных стилей игры
- Сложнее балансировать
- *Пример:* Cyberpunk 2077 - разные типы кибервара

**Вариант В: Глубокая детализация (10+ типов на категорию + подтипы)**
- Максимальное разнообразие и глубина
- Разные бренды/фракции = разные импланты
- Очень сложно балансировать
- *Пример:* Полная система, как в настольной игре Cyberpunk

**Вариант Г: Комбинация (базовый набор + расширяемый)**
- Базовый набор обязательных имплантов (для всех)
- Расширяемый набор (авторские + пользовательские через систему брендов)
- Гибкость для развития контента
- Интеграция с `equipment-matrix.md` (бренды и сигнатуры)

**Рекомендация:** Вариант Г (комбинация) - максимальная гибкость с базовой стабильностью

**Дополнительные вопросы:**
1. Нужны ли уникальные импланты из лора Cyberpunk (например, Сандевистан, кибердека)?
2. Должны ли быть импланты разных брендов (Arasaka, Militech, Zetatech) с разными характеристиками?
3. Как импланты связаны с системой способностей (`combat-abilities.md`)?

## Вопрос 2: Применение имплантов в бою

**Контекст:** Нужно определить, как конкретно каждый тип импланта влияет на боевую систему. Например, как боевые импланты улучшают стрельбу, как тактические помогают в стелсе.

**Варианты:**

**Вариант А: Прямое влияние (статы/модификаторы)**
- Импланты напрямую изменяют характеристики (точность +10%, урон +5%)
- Просто реализовать и балансировать
- Предсказуемое влияние
- *Пример:* Классическая система статов

**Вариант Б: Непрямое влияние (способности/эффекты)**
- Импланты дают новые способности или эффекты
- Более интересное разнообразие
- Сложнее балансировать
- *Пример:* Cyberpunk 2077 - импланты дают способности

**Вариант В: Комбинация прямого и непрямого**
- Базовые импланты = прямые модификаторы
- Продвинутые импланты = способности/эффекты
- Гибкость и разнообразие
- Прогрессия от простого к сложному

**Вариант Г: Полная интеграция с системами (статы + способности + синергии)**
- Импланты влияют через все системы: статы, способности, синергии с другими имплантами/экипировкой
- Максимальная глубина и разнообразие
- Очень сложно балансировать
- Интеграция с `combat-abilities.md` (способности), `equipment-matrix.md` (синергии брендов)

**Рекомендация:** Вариант Г (полная интеграция) - соответствует философии глубины игры

**Дополнительные вопросы:**
1. Должны ли импланты влиять на систему способностей (Q, E, R из VALORANT стиля)?
2. Как импланты связаны с кибердекой (cyberdeck) для хакерских способностей?
3. Нужны ли сетовые бонусы при использовании имплантов одного бренда/фракции?

## Вопрос 3: Связь имплантов с другими системами

**Контекст:** Импланты должны интегрироваться с системой способностей, боевой системой, экономикой, прокачкой. Нужно определить конкретные связи.

**Варианты:**

**Вариант А: Минимальная интеграция (только боевая система)**
- Импланты влияют только на боевые характеристики
- Просто реализовать
- Ограниченная глубина
- *Пример:* Упрощенная система

**Вариант Б: Умеренная интеграция (бой + способности)**
- Импланты влияют на бой и способности
- Больше разнообразия
- Нужна интеграция с `combat-abilities.md`

**Вариант В: Широкая интеграция (бой + способности + экономика)**
- Импланты влияют на бой, способности, экономику (торговля, крафт)
- Глубокая интеграция
- Нужна интеграция с `economy-crafting.md`, `economy-trading.md`

**Вариант Г: Полная интеграция (все системы)**
- Импланты интегрируются со всеми системами: бой, способности, экономика, прокачка, социальные механики
- Максимальная глубина и разнообразие
- Полная интеграция с существующими системами
- Интеграция: `combat-abilities.md`, `economy-crafting.md`, `progression/classes-overview.md`, `social/`

**Рекомендация:** Вариант Г (полная интеграция) - соответствует философии MMORPG с глубокими системами

**Дополнительные вопросы:**
1. Могут ли импланты влиять на социальные механики (репутация с фракциями, доступ к NPC)?
2. Как импланты связаны с системой прокачки (классы, навыки)?
3. Нужна ли интеграция с экономикой (торговля имплантами, крафт)?

## Вопрос 4: Примеры конкретных имплантов из лора Cyberpunk

**Контекст:** В лоре Cyberpunk есть много конкретных имплантов (Сандевистан, кибердека, киберглаза и т.д.). Нужно определить, какие импланты из лора включить и как они работают в игре.

**Варианты:**

**Вариант А: Только базовые импланты из лора**
- Включить только самые известные (Сандевистан, кибердека)
- Фокус на оригинальных механиках
- Ограниченная связь с лором

**Вариант Б: Широкий набор имплантов из лора**
- Включить большинство имплантов из лора Cyberpunk
- Глубокая связь с лором
- Больше работы по интеграции

**Вариант В: Все импланты из лора + авторские расширения**
- Включить все импланты из лора + создать авторские для MMORPG
- Максимальная связь с лором + гибкость
- Большой объем работы
- Интеграция с `equipment-matrix.md` (авторские бренды)

**Вариант Г: Комбинация (ядро из лора + авторские + пользовательские через бренды)**
- Ядро: основные импланты из лора Cyberpunk (Сандевистан, кибердека, киберглаза, киберруки и т.д.)
- Авторские: импланты для MMORPG механик (социальные, экономические)
- Пользовательские: через систему брендов (`equipment-matrix.md`)
- Максимальная гибкость и связь с лором

**Рекомендация:** Вариант Г (комбинация) - лучший баланс между лором и гибкостью

**Дополнительные вопросы:**
1. Какие конкретные импланты из лора Cyberpunk обязательно включить (например, из Cyberpunk 2077, настольной игры, аниме Edgerunners)?
2. Должны ли быть импланты разных брендов (Arasaka, Militech, Zetatech, Biotechnica) с разными характеристиками?
3. Как пользовательские импланты через бренды интегрируются с системой (модерация, баланс)?
