## Обсуждение

### Вселенная CYBERPUNK

**Решение:** Смешанный подход (гибридный)
- Лор CYBERPUNK из всех источников + авторский контент + реальная география
- Существующие корпорации из лора + новые авторские
- Реальные города из реального мира

### Временная линия лиг

**Решение:** Комбинация вариантов
- Линейная прогрессия А + нелинейное развитие Б + цикличная система Г
- Лиги прогрессируют линейно (2020-2090)
- Но развитие может быть нелинейным (некоторые технологии появляются раньше/позже)
- После 2090 возможен возврат к 2020, но с изменениями (цикличность)

### Фракции

**Решение:** Гибридная система
- Из оригинала + авторские + возможность создавать свои
- Фиксированные основные фракции + динамические второстепенные
- Корпорации, банды, организации из лора
- Возможность игрокам создавать свои фракции

### Локации

**Решение:**
- 20-30 городов на старт
- Все типы важны (богатые/бедные, корпоративные/бандитские, экстракт, PvP)
- Связь с региональными датацентрами

### Персонажи

**Решение:**
- 100+ важных NPC
- Живые NPC (как KENSHI, но живее)
- Игроки могут стать важными NPC (ФИКСЕРЫ, торговцы, политики)

### Симуляция

**Решение:** Многоуровневая концепция
- Всё вместе, раскрывается постепенно
- Вайб SAO (мир в киберпространстве за черным заслоном)
- Концепция симуляции как мета-концепция для объяснения лиг
