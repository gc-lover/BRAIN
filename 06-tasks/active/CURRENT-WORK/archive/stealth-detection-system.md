## Вопросы для проработки

### Вопрос 7.1: Система обнаружения для разных типов врагов

**Контекст:**
- Стелс с «разной системой обнаружения»; мир киберпанка: камеры, дроны, сканеры, ИИ, хакеры, импланты маскировки
- Экстракт/ПвЕ/ПвП требуют читаемых индикаторов и контр-игры

1) Каналы обнаружения
- A: Визуал + звук только
- Б: Визуал/звук + технологии (сканеры/тепловизор/эмп-сигнатуры)
- В: Визуал/звук/тех + сеть (слежение через камеры/сеть, ретрансляция между врагами)
- Г: Комбинация: все каналы, зависят от класса врага/зоны
Рекомендация: Г

2) Классы врагов и сильные стороны
- A: Охранник (визуал/звук)
- Б: Робот/дрон (тех-скан)
- В: Хакер/нетраннер (сеть/сигнатуры имплантов)
- Г: Комбинация: гибридные враги, профиль по роли/зоне
Рекомендация: Г

3) Влияние имплантов игрока
- A: Пассивные бонусы скрытности
- Б: Активные маскировки (кратковременные)
- В: Снижение сигнатур (тепло/эмп/сеть) ценой энергии/перегрева
- Г: Комбинация: пассивы + активки + снижение сигнатур с ценой
Рекомендация: Г (связать с энергобюджетом/перегревом)

4) Улучшение скрытности
- A: Навыки/перки
- Б: Экипировка/моды (тихие подошвы, глушители, ткань)
- В: Импланты (оптика/кожа/сердце)
- Г: Комбинация: все источники, с капами и конфликтами
Рекомендация: Г

5) Сетевые механики
- A: Камеры/датчики не связаны
- Б: Связанные — тревога передаётся локально
- В: Полноценная сеть зоны (ретрансляция, усиление обнаружения)
- Г: Комбинация: сеть по зонам/фракциям, узлы можно взломать (combat-hacking)
**✅ Решение (7.1):**
1) Каналы обнаружения — Г: Комбинация (по классам/зонам) + варианты из оригинальной настольной игры Cyberpunk и Cyberpunk 2077
2) Профили врагов — Г: Комбинация (гибридные враги по роли/зоне)
3) Влияние имплантов игрока — Г: Комбинация (пассивы + активки + снижение сигнатур с ценой энергии/перегрева)
4) Улучшение скрытности — Г: Комбинация (все источники, с капами/конфликтами) — пока оставляем так
5) Сети камер/датчиков — Г: Комбинация (сети по зонам/фракциям; узлы взламываемы) + локальные сети небольшие без выхода в глобальную сеть
