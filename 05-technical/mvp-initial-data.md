## Общее описание

Минимальный набор данных для первого запуска MVP текстовой версии NECPGAME. Определяет конкретные локации, NPC, предметы, квесты и другие данные, необходимые для функционирования игры.

**Цель:** Создать минимальный рабочий набор данных для проверки основных механик и игрового процесса.

## 1. Локации для MVP

### 1.1. Город для MVP

**Город:** Night City (главный город из лора Cyberpunk)

**Причина выбора:**
- Главный город из лора Cyberpunk
- Знакомый игрокам сеттинг
- Богатый лор и атмосфера
- Множество возможностей для развития

---

### 1.2. Районы Night City для MVP

**Минимум для MVP: 5 районов**

**1. Downtown (Корпоративный центр)**
- **Описание:** Центр корпоративной власти Night City. Небоскребы корпораций возвышаются над городом, неоновые вывески мигают на стенах зданий. Здесь располагаются офисы Arasaka, Militech и других мегакорпораций.
- **Тип:** Корпоративный район
- **Опасность:** Низкая (контролируется корпорациями и NCPD)
- **Минимальный уровень:** 1
- **Фракции:** Arasaka, Militech, NCPD
- **NPC:** Корпоративные торговцы, охранники, офисные работники

**2. Watson (Индустриальный район)**
- **Описание:** Индустриальный район с заброшенными заводами и складами. Здесь смешиваются рабочие кварталы и криминальные зоны. Много торговцев и незаконных операций.
- **Тип:** Индустриальный/криминальный район
- **Опасность:** Средняя
- **Минимальный уровень:** 1
- **Фракции:** Valentinos, Maelstrom, Tyger Claws
- **NPC:** Торговцы, фиксеры, бандиты

**3. Westbrook (Богатый район)**
- **Описание:** Район для богатых жителей Night City. Элитные жилые комплексы, дорогие магазины, роскошные клубы. Здесь живут корпоративные руководители и влиятельные персоны.
- **Тип:** Жилой/торговый район
- **Опасность:** Низкая (контролируется корпорациями)
- **Минимальный уровень:** 3
- **Фракции:** Arasaka, Militech, корпоративные клиенты
- **NPC:** Элитные торговцы, корпоративные представители, охранники

**4. Santo Domingo (Рабочий район)**
- **Описание:** Рабочий район с промышленными зонами и жилыми кварталами. Здесь живут рабочие и низкоуровневые жители. Много торговых точек и базовых услуг.
- **Тип:** Рабочий/жилой район
- **Опасность:** Низкая-Средняя
- **Минимальный уровень:** 1
- **Фракции:** 6th Street, Valentinos
- **NPC:** Базовые торговцы, рабочие, курьеры

**5. Heywood (Бандитский район)**
- **Описание:** Район под контролем банд. Уличная культура, граффити на стенах, постоянные конфликты между бандами. Опасное место для новичков.
- **Тип:** Бандитский район
- **Опасность:** Высокая
- **Минимальный уровень:** 5
- **Фракции:** Valentinos, 6th Street, Maelstrom
- **NPC:** Бандиты, нелегальные торговцы, фиксеры

### 1.3. Связи между локациями

**Структура связей:**

```
Downtown ←→ Watson (связаны)
Downtown ←→ Westbrook (связаны)
Watson ←→ Santo Domingo (связаны)
Santo Domingo ←→ Heywood (связаны)
Watson ←→ Heywood (связаны, опасный путь)
```

**Правила перемещения:**
- Перемещение между соседними районами занимает 5-10 минут игрового времени
- Перемещение между несоседними районами требует прохождения через промежуточные районы
- Перемещение в опасные районы (Heywood) требует подтверждения
- Перемещение может стоить денег (опционально, зависит от способа перемещения)

## 9. Вопросы для проработки

### Q1: Количество локаций

**Вопрос:** Сколько локаций достаточно для MVP?

**Решение (гибридный подход):**
- Минимум: 5 районов в Night City
- Каждый район имеет 2-3 точки интереса (здания, улицы)
- Итого: 10-15 локаций для полноценного игрового процесса
- Можно начать с 5 районов и расширять по мере разработки

---

### Q2: Количество NPC

**Вопрос:** Сколько NPC достаточно для MVP?

**Решение (гибридный подход):**
- Минимум: 10 NPC (распределены по локациям)
- Каждый NPC имеет базовую функциональность (торговля, квесты, информация)
- Можно расширять по мере разработки
- Для полноценного игрового процесса нужно 20-30 NPC

---

### Q3: Количество предметов

**Вопрос:** Сколько предметов достаточно для MVP?

**Решение (гибридный подход):**
- Минимум: 15 предметов (оружие, броня, медикаменты, ресурсы)
- Базовые предметы для разных уровней (1-7)
- Можно расширять по мере разработки
- Для полноценного игрового процесса нужно 50-100 предметов

---

### Q4: Количество квестов

**Вопрос:** Сколько квестов достаточно для MVP?

**Решение (гибридный подход):**
- Минимум: 5 квестов (разных типов и уровней)
- Простые квесты для новичков (уровень 1-2)
- Средние квесты для опытных (уровень 3-5)
- Можно расширять по мере разработки
- Для полноценного игрового процесса нужно 20-30 квестов
