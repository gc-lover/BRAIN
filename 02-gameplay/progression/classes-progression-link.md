## Связь классов с системой прокачки

### Механика связи классов и прокачки

**Решение (2025-11-03):** Гибридная система (Вариант А + Вариант Б + общие навыки)
- **Классы как стартовые бонусы:** Класс дает стартовые бонусы к определенным навыкам
- **Классы определяют доступные навыки:** Класс открывает доступ к определенным веткам навыков (классовым)
- **Общие навыки:** Есть общие навыки, которые может прокачивать любой класс
- **Свободная прокачка:** Дальше игрок может прокачиваться свободно в любом направлении (но в рамках доступных веток)

**Структура навыков:**
- **Классовые навыки:** Уникальные ветки навыков для каждого класса
- **Общие навыки:** Универсальные навыки, доступные всем классам
- **"Свой путь":** Может прокачивать только общие навыки (без классовых веток)

**Механика прокачки:**
- Класс дает стартовые бонусы к своим навыкам
- Класс открывает доступ к уникальным веткам навыков
- Общие навыки доступны всем классам и могут прокачиваться свободно
- Классовые навыки прокачиваются быстрее/эффективнее для своего класса
- Игрок может развиваться в любом направлении в рамках доступных веток
