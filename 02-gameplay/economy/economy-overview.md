## Концепция экономической системы

### Важность экономики

**Почему важна:**
- Глобальная экономика - один из основных столпов игры
- MMORPG требует глубокой экономической системы
- Экономика влияет на все аспекты игры (торговля, крафт, социальные структуры)
- Живой мир требует живую экономику (как в KENSHI и EVE Online)
- Экономические взаимодействия между игроками, фракциями, корпорациями

**Источники вдохновения:**
- **EVE Online** - глобальная экономика, политика, торговля, рынок игроков
- **RIMWORLD** - глобальная экономика, торговля, взаимодействия фракций
- **KENSHI** - живой мир, экономика развивается сама по себе
- **WOW** - система торговли и аукционов
- **MINECRAFT** - система крафта и торговли
- **CYBERPUNK 2077** - экономика в киберпанк сеттинге

## Вопрос 1: Тип экономической системы

**Как должна работать экономическая система?**

**Контекст:**

**EVE Online экономика:**
- Полностью игроко-управляемая экономика
- Рынок игроков (торговля между игроками)
- Производство игроками (крафт, добыча)
- Региональные рынки и торговые маршруты
- Влияние фракций на экономику

**WOW экономика:**
- Смешанная экономика (NPC торговцы + рынок игроков)
- Аукцион дом для торговли
- Крафт и профессии
- Валюты (золото, валюта PvP/PvE)

**RIMWORLD экономика:**
- Региональная торговля между фракциями
- Разные фракции имеют разные ресурсы и цены
- Влияние отношений на торговлю
- Динамические цены

**Вопросы:**

1. **Кто управляет экономикой?**
   - Вариант А: Полностью игроко-управляемая экономика (EVE Online стиль)
     - Рынок игроков определяет цены
     - Игроки создают и продают всё
     - Минимальное вмешательство NPC
     - *Плюсы:* Максимальная свобода, глубокая экономика
     - *Минусы:* Может быть сложно для новичков
   
   - Вариант Б: Смешанная экономика (WOW стиль)
     - NPC торговцы для базовых товаров
     - Рынок игроков для редких товаров
     - Баланс между NPC и игроками
     - *Плюсы:* Доступность для всех, стабильность
     - *Минусы:* Меньше свободы для игроков
   
   - Вариант В: Региональная экономика (RIMWORLD стиль)
     - Разные регионы/фракции имеют разные ресурсы и цены
     - Торговля между регионами/фракциями
     - Влияние отношений на торговлю
     - *Плюсы:* Реализм, глубина взаимодействий
     - *Минусы:* Сложнее балансировать
   
   - Вариант Г: Гибридная система (комбинация всех вариантов)
     - Базовые товары у NPC (стабильность)
     - Редкие товары на рынке игроков (свобода)
     - Региональные различия в ресурсах и ценах (реализм)
     - *Плюсы:* Максимальная гибкость
     - *Минусы:* Сложнее реализовать

   **Вопрос:** Какой вариант? (А/Б/В/Г)

2. **Как работает рынок?**
   - Вариант А: Глобальный рынок (один рынок для всех)
     - Все игроки видят все предложения
     - Единые цены
     - *Плюсы:* Простота и доступность
     - *Минусы:* Меньше разнообразия
   
   - Вариант Б: Региональные рынки (разные рынки для регионов)
     - Разные рынки в разных регионах/городах
     - Разные цены в разных регионах
     - Торговые маршруты между регионами
     - *Плюсы:* Реализм, возможности для торговли
     - *Минусы:* Сложнее для игроков
   
   - Вариант В: Фракционные рынки (разные рынки для фракций)
     - Разные рынки для разных фракций
     - Влияние репутации на доступ к рынкам
     - Торговля между фракциями
     - *Плюсы:* Глубина социальных взаимодействий
     - *Минусы:* Может ограничивать игроков
   
   - Вариант Г: Комбинация (глобальный + региональные + фракционные)
     - Глобальный рынок для общих товаров
     - Региональные рынки для местных товаров
     - Фракционные рынки для эксклюзивных товаров
     - *Плюсы:* Максимальная гибкость
     - *Минусы:* Очень сложно реализовать

   **Вопрос:** Какой вариант? (А/Б/В/Г)

3. **Как определяются цены?**
   - Вариант А: Динамические цены (спрос и предложение)
     - Цены меняются в зависимости от спроса и предложения
     - Игроки влияют на цены своими действиями
     - *Плюсы:* Реализм, глубина экономики
     - *Минусы:* Может быть нестабильно
   
   - Вариант Б: Фиксированные цены (установленные NPC/системой)
     - NPC устанавливают цены
     - Стабильные цены для базовых товаров
     - *Плюсы:* Стабильность, предсказуемость
     - *Минусы:* Меньше глубины
   
   - Вариант В: Гибридная система (базовые цены + динамика)
     - Базовые цены устанавливаются NPC/системой
     - Игроки могут влиять на цены через торговлю
     - Динамика для редких товаров
     - *Плюсы:* Баланс между стабильностью и свободой
     - *Минусы:* Сложнее балансировать

   **Вопрос:** Какой вариант? (А/Б/В)

## Вопрос 2: Валюты и ресурсы

**Какие валюты и ресурсы должны быть в игре?**

**Контекст:**

**WOW валюты:**
- Золото (основная валюта)
- Валюта PvP (бонусы за PvP)
- Валюта PvE (бонусы за рейды)

**EVE Online валюты:**
- ISK (основная валюта)
- Разные типы ресурсов (руды, материалы, компоненты)

**CYBERPUNK 2077 валюты:**
- Евродоллары (основная валюта)
- Ресурсы для крафта

**Вопросы:**

1. **Типы валют:**
   - Вариант А: Одна валюта (простота)
     - Одна основная валюта для всего
     - *Плюсы:* Простота и доступность
     - *Минусы:* Меньше разнообразия
   
   - Вариант Б: Несколько валют (разные типы)
     - Основная валюта (золото, евродоллары)
     - Валюта PvP (бонусы за PvP)
     - Валюта PvE (бонусы за рейды/контент)
     - Валюта фракций (влияние на фракции)
     - *Плюсы:* Глубина, разнообразие
     - *Минусы:* Может быть сложно управлять
   
   - Вариант В: Региональные валюты (разные валюты для регионов)
     - Разные валюты в разных регионах/странах
     - Обменные курсы между валютами
     - *Плюсы:* Реализм, возможности для торговли
     - *Минусы:* Сложнее для игроков

   **Вопрос:** Какой вариант? (А/Б/В)

2. **Типы ресурсов:**
   - Базовые ресурсы (руды, материалы)
   - Редкие ресурсы (специальные материалы)
   - Ресурсы для крафта (компоненты)
   - Ресурсы для имплантов (кибервар материалы)
   - Ресурсы для фракций (влияние на фракции)
   - Другие типы?

   **Вопрос:** Какие типы ресурсов нужны? (опишите или выберите)

3. **Как добываются ресурсы?**
   - Вариант А: Добыча в экстракт-зонах (TARKOV стиль)
     - Выход в зоны для добычи ресурсов
     - Риск потери при смерти
     - *Плюсы:* Риск/награда, хардкорность
     - *Минусы:* Может быть сложно
   
   - Вариант Б: Добыча в открытом мире (WOW стиль)
     - Ресурсы добываются везде в открытом мире
     - Нет риска потери
     - *Плюсы:* Доступность для всех
     - *Минусы:* Меньше напряжения
   
   - Вариант В: Производство игроками (EVE Online стиль)
     - Игроки производят ресурсы через крафт/производство
     - Добыча исходных материалов
     - *Плюсы:* Глубина экономики
     - *Минусы:* Может быть сложно
   
   - Вариант Г: Комбинация всех способов
     - Добыча в экстракт-зонах (редкие ресурсы)
     - Добыча в открытом мире (базовые ресурсы)
     - Производство игроками (обработанные ресурсы)
     - *Плюсы:* Максимальная гибкость
     - *Минусы:* Сложнее реализовать

   **Вопрос:** Какой вариант? (А/Б/В/Г)

## Вопрос 3: Торговля

**Как должна работать торговля?**

**Контекст:**

**EVE Online торговля:**
- Торговля между игроками через рынок
- Торговые маршруты между регионами
- Влияние фракций на торговлю
- Спекуляция и торговые операции

**WOW торговля:**
- Аукцион дом для торговли
- Торговля между игроками напрямую
- NPC торговцы для базовых товаров

**RIMWORLD торговля:**
- Торговля с другими фракциями
- Влияние отношений на торговлю
- Динамические цены в зависимости от отношений

**Вопросы:**

1. **Типы торговли:**
   - Вариант А: Рынок игроков (EVE Online стиль)
     - Игроки создают ордера на покупку/продажу
     - Рыночная система с поиском товаров
     - *Плюсы:* Глубина, свобода
     - *Минусы:* Может быть сложно для новичков
   
   - Вариант Б: Аукцион дом (WOW стиль)
     - Аукцион дом для продажи товаров
     - Простая система торговли
     - *Плюсы:* Простота и доступность
     - *Минусы:* Меньше гибкости
   
   - Вариант В: Прямая торговля между игроками
     - Торговля напрямую между игроками
     - Обмен товарами
     - *Плюсы:* Непосредственное взаимодействие
     - *Минусы:* Может быть неудобно
   
   - Вариант Г: Комбинация всех типов
     - Рынок игроков для масштабной торговли
     - Аукцион дом для быстрой торговли
     - Прямая торговля для особых случаев
     - *Плюсы:* Максимальная гибкость
     - *Минусы:* Сложнее реализовать

   **Вопрос:** Какой вариант? (А/Б/В/Г)

   - Влияние репутации на цены (скидки/надбавки)
   - Доступ к фракционным рынкам по репутации
   - Торговые бонусы для союзников
   - Ограничения на торговлю с врагами
   - Торговые войны между фракциями
   - Другие механики?

   **Вопрос:** Какие механики влияния социальных структур нужны? (опишите)

3. **Торговые маршруты:**
   - Вариант А: Нет торговых маршрутов (торговля только локально)
     - Торговля только в одном месте
     - *Плюсы:* Простота
     - *Минусы:* Меньше глубины
   
   - Вариант Б: Торговые маршруты между регионами (EVE Online стиль)
     - Торговля между разными регионами
     - Разные цены в разных регионах
     - Возможность заработать на перевозке товаров

     - *Минусы:* Может быть сложно
   
   - Вариант В: Торговые маршруты через фракции (RIMWORLD стиль)
     - Торговля с другими фракциями
     - Влияние отношений на торговлю

     - Рецепты открываются при прокачке
     - *Плюсы:* Прогрессия
     - *Минусы:* Может ограничивать
   
   - Вариант В: Рецепты как лут/находки

- Источники вдохновения: FORTNITE, CS, DOTA2 (монетизация через скины)

**Вопросы:**

1. **Создание контента игроками:**
   - Создание скинов для оружия/брони
   - Создание экипировки/предметов
