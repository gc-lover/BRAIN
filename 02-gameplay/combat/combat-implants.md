### Важность имплантов

Импланты в NECPGAME являются важной частью киберпанк сеттинга и боевой системы. Они определяют способности персонажа и создают разнообразие билдов.

## Основные вопросы для проработки

### 1. Типы имплантов для боя

**Боевые импланты:**
- Улучшение стрельбы (точность, урон)
- Боевые способности (дополнительные атаки)
- Улучшение оружия (установка модов на оружие через импланты)
- *Пример:* Киберимпланты для рук в CYBERPUNK 2077

**Тактические импланты:**
- Сканирование врагов (видение через стены, информация о здоровье)
- Хакерство (взлом врагов, отключение их имплантов)
- Стелс (скрытность, обнаружение)
- *Пример:* Кибермод для глаз (сканер) в CYBERPUNK 2077

**Защитные импланты:**
- Броня (увеличение защиты)
- Восстановление (регенерация здоровья)
- Иммунитет (защита от эффектов)
- *Пример:* Субдермальная броня в CYBERPUNK 2077

**Двигательные импланты:**
- Улучшение движения (скорость, прыжки, бег)
- Паркур способности
- Уклонение (даш, уворот)
- *Пример:* Нейро-импланты для ног в CYBERPUNK 2077

**Операционные системы (OS):**
- Системы для управления способностями
- Хакерские системы
- Боевые системы
- *Пример:* Сандвистан, кибердека в CYBERPUNK 2077

### 2. Ограничения имплантов

**✅ Решение:** Комбинация ограничений (слоты + совместимость + общий лимит + лимит по человечности + энергетический лимит)

#### 2.1 Какие ограничения на импланты?

**✅ Решение:** Комбинация ограничений (слоты + совместимость + общий лимит)

- **Система слотов (разные типы слотов):**
  - Разные типы слотов для разных имплантов
  - Боевые слоты, тактические слоты, защитные слоты, двигательные слоты, OS слот
  - Каждый тип импланта требует свой слот
  - Логика и баланс установки имплантов

- **Совместимость имплантов (некоторые несовместимы):**
  - Некоторые импланты несовместимы друг с другом
  - Установка несовместимых = больше потери человечности или невозможна
  - Реализм и тактический выбор при планировании билда

- **Общий лимит на количество имплантов:**
  - Максимум X имплантов всего (независимо от типа)
  - Дополнительное ограничение сверх системы слотов
  - Баланс между количеством и качеством имплантов

- **Влияние класса игрока:**
  - Класс игрока может влиять на доступные слоты/лимиты
  - Разные классы = разные возможности (например, Netrunner = больше тактических слотов)

- **Увеличение лимитов:**
  - Можно увеличить лимиты имплантов (прокачка, специальные импланты)
  - Прогрессия через игровой процесс

- **Визуальное проявление имплантов:**
  - Импланты влияют на внешность персонажа (как в CYBERPUNK 2077)
  - Визуальные изменения в зависимости от установленных имплантов

**Статус:** Решено

#### 2.2 Как работает лимит имплантов?

**✅ Решение:** Комбинация факторов (базовый лимит + класс/прокачка + человечность)

- **Базовый лимит для всех игроков:**
  - Все игроки имеют одинаковый базовый лимит имплантов
  - Например: базовый максимум 5-10 имплантов для всех
  - Справедливость для всех игроков

- **Бонусы от класса/прокачки (увеличение лимита):**
  - Разные классы = разные дополнительные лимиты имплантов
  - Прокачка навыков = увеличение лимита
  - Некоторые классы (например, Netrunner, Medtech) могут иметь бонусы к лимиту
  - Персонализация и прогрессия через игровой процесс

- **Влияние человечности (снижение лимита при потере):**
  - Чем больше человечности = больше лимит имплантов (бонус)
  - Потеря человечности = снижение лимита
  - Киберпсихоз = критические ограничения (снижение лимита до минимума)
  - Связано с системой киберпсихоза, логично и реалистично

- **Временное превышение лимита:**
  - Можно временно превысить лимит (с последствиями для человечности)
  - Превышение = увеличенная скорость потери человечности
  - Риск киберпсихоза при превышении лимита

- **Влияние качества имплантов:**
  - Качественные импланты = меньше "вес" лимита (опционально)
  - Дешевые/низкокачественные импланты = больше "вес" лимита (опционально)
  - Баланс между качеством и количеством

**Статус:** Решено

#### 2.3 Как работает энергетический лимит?

**✅ Решение:** Гибридная система (общий пул + индивидуальные ограничения)

- **Общий энергетический пул для всех имплантов:**
  - Все импланты используют общий энергетический пул
  - Использование способностей = расход энергии из общего пула
  - Тактический выбор (какие способности использовать при ограниченной энергии)

- **Индивидуальные ограничения для сложных имплантов:**
  - Сложные импланты требуют больше энергии (индивидуальные требования)
  - Некоторые импланты могут иметь свою энергию (особенно сложные, например, кибердека)
  - Баланс между простыми и сложными имплантами

- **Увеличение энергетического пула:**
  - Можно увеличить энергетический пул (прокачка, импланты)
  - Прогрессия через игровой процесс
  - Специальные импланты для увеличения энергетического пула

- **Влияние энергии на киберпсихоз:**
  - Перегрузка энергии = больше потери человечности (опционально)
  - Использование слишком много энергии одновременно = риск киберпсихоза
  - Баланс между мощью и риском

- **Восстановление энергии:**
  - Время (пассивное восстановление)
  - Предметы (энергетические батареи, импланты)
  - Навыки (прокачка навыков для увеличения скорости восстановления)
  - Разные методы для разных ситуаций

**Статус:** Решено

### 2.1 Детали киберпсихоза

**Исследованные источники:**
- **Cyberpunk 2077 (игра):** Киберпсихоз представлен через побочные миссии, NPC теряют рассудок из-за избытка имплантов. В базовой игре игрок не подвержен риску (критика), в Phantom Liberty планировалась механика для игрока.
- **Cyberpunk настольная игра:** Каждый имплант снижает уровень человечности (Humanity). При достижении нулевого уровня человечности персонаж становится киберпсихотиком - теряет контроль, становится агрессивным и опасным.
- **Cyberpunk Edgerunners (аниме):** Постепенная потеря рассудка из-за чрезмерного использования имплантов. Нарастающие симптомы, трагические последствия. Демонстрирует потерю человечности и разрушительные последствия.

**✅ Решение:** Полная детальная система киберпсихоза с прогрессией по стадиям

#### 2.1.1 Система человечности (Humanity)

**✅ Решение:** Гибридная система (базовая потеря + дополнительные факторы)
- **Базовая потеря человечности при установке импланта:**
  - Каждый имплант снижает человечность при установке
  - Разные типы имплантов = разная базовая потеря
  - Боевые импланты = больше базовой потери
  - Бытовые импланты = меньше базовой потери
- **Дополнительные факторы влияют на скорость потери:**
  - **Тип импланта:** Боевые = больше потери, бытовые = меньше потери
  - **Количество имплантов:** Чем больше, тем быстрее теряется человечность
  - **Использование способностей:** Особенно с потреблением здоровья/энергии = дополнительная потеря
  - **Время адаптации:** Меньше потери при адаптации (время с имплантами снижает скорость потери)
  - **Качество имплантов:** Качественные импланты = меньше потери, дешевые/низкокачественные = больше потери
  - **Качество риппера (установщика):** Профессиональный риппер = меньше потери при установке, любитель/неопытный = больше потери
  - **Класс игрока:** Некоторые классы (например, Medtech) могут иметь бонусы к адаптации или снижению потери человечности
  - **Совместимость имплантов:** Несовместимые импланты = больше потери человечности
  - **Интенсивность использования:** Частое использование имплантов = ускорение потери человечности

#### 2.1.2 Симптомы и прогрессия киберпсихоза

**✅ Решение:** Комбинация (стадии + симптомы)
- **Стадии киберпсихоза:**
  - **Ранняя стадия:** Начальные симптомы, легкие последствия
  - **Средняя стадия:** Усиленные симптомы, умеренные последствия
  - **Поздняя стадия:** Серьезные симптомы, тяжелые последствия
  - **Киберпсихоз:** Полная потеря контроля, критическое состояние
- **Симптомы на каждой стадии:**
  - **Ранняя стадия:** Раздражение, агрессивность, легкая потеря эмпатии
  - **Средняя стадия:** Вспышки агрессии, потеря контроля, паранойя, отчуждение
  - **Поздняя стадия:** Галлюцинации, серьезная потеря контроля, неконтролируемая агрессия
  - **Киберпсихоз:** Полная потеря контроля, психоз, критические эффекты
- **Прогрессия:** Симптомы нарастают постепенно, не сразу
- **Влияние на игровой процесс:** Каждая стадия имеет набор симптомов, которые влияют на игровой процесс по-разному

#### 2.1.3 Последствия киберпсихоза

**✅ Решение:** Прогрессия последствий по стадиям
- **Ранняя стадия (простые последствия):**
  - Легкие негативные эффекты (снижение некоторых характеристик)
  - Редкие случайные действия (легкая потеря контроля)
  - Визуальные эффекты (легкие искажения)
  - Минимальное влияние на игровой процесс
- **Средняя стадия (умеренные последствия):**
  - Потеря контроля в бою (временная, случайная агрессия)
  - Серьезные негативные эффекты (снижение характеристик, потеря точности)
  - Приступы агрессии (неконтролируемые действия против врагов/союзников)
  - Визуальные искажения (галлюцинации, отчуждение)
  - Влияние на социальные взаимодействия (другие игроки видят симптомы)
- **Поздняя стадия (серьезные последствия):**
  - Частая потеря контроля (временная потеря контроля над персонажем)
  - Критические эффекты (значительное снижение характеристик, потеря способностей)
  - Приступы неконтролируемой агрессии (атака на союзников, потеря контроля)
  - Серьезные визуальные искажения (галлюцинации, психоз)
  - Серьезное влияние на социальные взаимодействия (репутация, доступ к контенту)
- **Киберпсихоз (критические последствия):**
  - Полная потеря контроля (временная потеря контроля над персонажем, может атаковать союзников)
  - Критические негативные эффекты (значительное снижение всех характеристик)
  - Неконтролируемая агрессия (атака на всех вокруг, включая союзников)
  - Критические визуальные эффекты (полный психоз, потеря связи с реальностью)
  - Критическое влияние на социальные взаимодействия (другие игроки могут избегать, влияние на репутацию)
  - Возможность лечения/восстановления (не постоянная смерть, но серьезные последствия)
  - На критической стадии: два пути решения - смерть персонажа или помощь других игроков (обезвреживание и принудительная терапия)
- **Прогрессия:** Последствия нарастают постепенно от легких до критических по мере развития стадий

#### 2.1.4 Управление киберпсихозом

**✅ Решение:** Комбинация способов (профилактика, лечение, управление, адаптация, препараты, терапия)
- **Профилактика:**
  - Балансировка количества/типа имплантов (не устанавливать слишком много сразу)
  - Выбор качественных имплантов (меньше потери человечности)
  - Выбор профессионального риппера (меньше потери при установке)
  - Планирование совместимости имплантов (избегать несовместимых)
- **Лечение:**
  - Восстановление человечности у врача (дорого, требует времени)
  - Удаление имплантов восстанавливает человечность (но потеря способностей)
  - Специальное лечение киберпсихоза (дорогое, долгое, но эффективное)
- **Управление:**
  - Контроль использования способностей (особенно с потреблением здоровья/энергии)
  - Снижение интенсивности использования имплантов (реже использовать = медленнее потеря)
  - Выбор стратегии использования (баланс между мощью и риском)
- **Адаптация:**
  - Время с имплантами = постепенная адаптация (меньше потери человечности)
  - Использование имплантов = адаптация (привыкание снижает скорость потери)
  - Прокачка навыков = лучше адаптация (некоторые навыки улучшают адаптацию)
- **Препараты:**
  - Временное снижение симптомов киберпсихоза (не лечит, только управляет симптомами)
  - Препараты для снижения потери человечности (временный эффект)
  - Разные типы препаратов для разных стадий
- **Терапия:**
  - Психологическая терапия (долгосрочное решение, снижает скорость потери)
  - Специализированная терапия киберпсихоза (долгая, дорогая, но эффективная)
  - Регулярная терапия = профилактика и управление

#### 2.1.5 Киберпсихоз в MMORPG контексте

**✅ Решение:** Полная интеграция в MMORPG с учетом всех аспектов
- **Потеря контроля в MMORPG:**
  - **Ранняя стадия:** Только временная потеря контроля (короткие приступы)
  - **Средняя стадия:** Временная потеря с увеличенной длительностью
  - **Поздняя стадия:** Серьезная потеря контроля (требуется помощь других игроков)
  - **Киберпсихоз:** Критическая потеря контроля (два пути: смерть или помощь других игроков через обезвреживание и принудительную терапию)
- **Влияние на социальные взаимодействия:**
  - **Ранняя стадия:** Минимальное влияние (легкие визуальные симптомы, небольшое влияние на репутацию)
  - **Средняя стадия:** Умеренное влияние (игроки видят симптомы, влияние на репутацию, некоторые ограничения)
  - **Поздняя стадия:** Серьезное влияние (игроки избегают, значительное влияние на репутацию, ограничения доступа)
  - **Киберпсихоз:** Критическое влияние (сильное влияние на все социальные взаимодействия, серьезные ограничения)
- **Киберпсихоз как игровая механика (рискованная мощь):**
  - Высокий риск киберпсихоза = временные бонусы к мощности (при приступах агрессии)
  - Но также и серьезные недостатки (потеря контроля, негативные эффекты)
  - Игрок может управлять балансом (больше мощности = больше риска)
  - Лечение снижает риск, но и мощность
- **Балансировка с игровым опытом:**
  - Последствия значимы, но управляемы
  - Лечение доступно, но требует ресурсов
  - Не блокирует полностью игровой процесс, но создает проблемы
  - Важность баланса между мощью и риском
- **Визуальные и аудиальные проявления:**
  - **Ранние стадии:** Только для игрока (приватность)
  - **Средние стадии:** Частично видны другим (легкие визуальные эффекты)
  - **Поздние стадии:** Видны всем (серьезные визуальные/аудиальные эффекты)
  - **Киберпсихоз:** Критические эффекты для всех (очень явные визуальные индикаторы)
- **Киберпсихоз в PvP и PvE контексте:**
  - **В PvE:** Полные последствия киберпсихоза
  - **В PvP:** Уменьшенные последствия с опциями (игрок может выбирать риск)
  - **В безопасных зонах:** Минимальные последствия
  - **В киберпространстве:** Специальные правила для виртуального пространства
- **Восстановление после киберпсихоза:**
  - **Ранние стадии:** Полное восстановление возможно
  - **Средние стадии:** Частичное восстановление с легкими шрамами (-5% максимальной человечности)
  - **Поздние стадии:** Частичное восстановление с серьезными шрамами (-15% максимальной человечности)
  - **Киберпсихоз:** Восстановление возможно, но с критическими шрамами (-30% максимальной человечности)
  - Множественные случаи киберпсихоза = накопление шрамов
