## Концепция имплантов

### Важность имплантов

Импланты в NECPGAME являются важной частью киберпанк сеттинга и боевой системы. Они определяют способности персонажа и создают разнообразие билдов.

**Уже определено:**
- Импланты дают способности
- Киберпанк стиль (импланты как часть персонажа)
- Вдохновение: CYBERPUNK 2077 - кибервар и импланты

---

## Основные вопросы для проработки

### 1. Типы имплантов для боя

**Боевые импланты:**
- Улучшение стрельбы (точность, урон)
- Боевые способности (дополнительные атаки)
- Улучшение оружия (установка модов на оружие через импланты)
- *Пример:* Киберимпланты для рук в CYBERPUNK 2077

**Тактические импланты:**
- Сканирование врагов (видение через стены, информация о здоровье)
- Хакерство (взлом врагов, отключение их имплантов)
- Стелс (скрытность, обнаружение)
- *Пример:* Кибермод для глаз (сканер) в CYBERPUNK 2077

**Защитные импланты:**
- Броня (увеличение защиты)
- Восстановление (регенерация здоровья)
- Иммунитет (защита от эффектов)
- *Пример:* Субдермальная броня в CYBERPUNK 2077

**Двигательные импланты:**
- Улучшение движения (скорость, прыжки, бег)
- Паркур способности
- Уклонение (даш, уворот)
- *Пример:* Нейро-импланты для ног в CYBERPUNK 2077

**Операционные системы (OS):**
- Системы для управления способностями
- Хакерские системы
- Боевые системы
- *Пример:* Сандвистан, кибердека в CYBERPUNK 2077

**Вопросы:**
1. Какие типы имплантов приоритетны для боя?
2. Могут ли импланты комбинироваться друг с другом?
3. Должны ли импланты быть визуально видны на персонаже?

### 2. Ограничения имплантов

**Лимит на количество имплантов:**
- Игрок может установить ограниченное количество
- Слоты для имплантов (например, 5-10 слотов)
- Разные импланты занимают разное количество слотов
- *Плюсы:* Простота и баланс
- *Минусы:* Может быть ограничивающим

**Лимит на "киберпсихоз":**
- Слишком много имплантов = проблемы
- "Толерантность" к имплантам растет с прокачкой
- Превышение лимита = негативные эффекты (снижение характеристик, потеря контроля)
- Киберпсихоз как игровая механика риска/награды
- *Пример:* CYBERPUNK 2077 киберпсихоз
- *Плюсы:* Лорно, реалистично, добавляет глубину и выбор
- *Минусы:* Сложнее система

**Детали киберпсихоза:**

**Симптомы киберпсихоза:**
- Снижение характеристик
- Потеря контроля над персонажем (временная)
- Визуальные искажения
- Агрессивное поведение (неконтролируемое)
- Другие негативные эффекты

**Прогрессия киберпсихоза:**
- Низкий уровень: Минимальные эффекты
- Средний уровень: Умеренные эффекты
- Высокий уровень: Серьезные эффекты
- Критический уровень: Опасные эффекты (потеря контроля)

**Управление киберпсихозом:**
- Вариант А: Только снижение количества имплантов
- Вариант Б: Лечение/терапия для снижения киберпсихоза
- Вариант В: Специальные импланты для защиты от киберпсихоза
- Вариант Г: Комбинация способов управления

**Вопросы:**
1. Должен ли киберпсихоз быть постоянным или временным?
2. Могут ли игроки полностью вылечить киберпсихоз?
3. Должен ли киберпсихоз влиять на игровой процесс (например, на бой)?
4. Могут ли игроки использовать киберпсихоз как преимущество? (например, усиление агрессии)

**Примечание:** Обсудим позже

**Энергетический лимит:**
- Импланты потребляют энергию
- Лимит энергии ограничивает количество активных имплантов
- Можно улучшать энергоемкость
- *Плюсы:* Тактическое управление ресурсами
- *Минусы:* Дополнительная сложность

**Комбинация ограничений:**
- Количество слотов + толерантность + энергия
- Разные импланты имеют разные требования
- *Плюсы:* Глубокая система
- *Минусы:* Очень сложно

**Варианты:**
- **Вариант А:** Лимит на количество имплантов (слоты)
- **Вариант Б:** Лимит на киберпсихоз (толерантность)
- **Вариант В:** Энергетический лимит
- **Вариант Г:** Комбинация ограничений
- **Вариант Д:** Свой вариант

### 3. Визуальное проявление имплантов

**Варианты:**

**Вариант А: Импланты видны на персонаже**
- Визуально видны все импланты
- Часть кастомизации персонажа
- *Плюсы:* Визуальная обратная связь, кастомизация
- *Минусы:* Может быть слишком много визуальных эффектов

**Вариант Б: Импланты скрыты (но эффекты видны)**
- Импланты не видны визуально
- Но эффекты от них видны (неон, свечение)
- *Плюсы:* Чище внешний вид
- *Минусы:* Меньше визуальной обратной связи

**Вариант В: Частично видны**
- Некоторые импланты видны (руки, глаза)
- Другие скрыты (внутренние)
- *Плюсы:* Баланс между визуалом и реализмом
- *Минусы:* Нужно определять какие видны

**Статус:** Требует решения

2. Могут ли игроки видеть импланты других игроков (для тактики)?
3. Должны ли импланты иметь визуальные эффекты при использовании?

### 4. Получение и улучшение имплантов

**Вопросы:**

1. **Как получают импланты?**
   - Вариант А: Покупка у NPC (ФИКСЕРЫ, торговцы)
   - Вариант Б: Добыча в экстракт-зонах
   - Вариант В: Крафт (создание имплантов)
   - Вариант Г: Комбинация всех способов

2. **Можно ли улучшать импланты?**
   - Вариант А: Нет, только замена на лучший
   - Вариант Б: Да, можно апгрейдить существующие
   - Вариант В: Комбинация (улучшение + замена)

3. **Редкость имплантов:**
   - Вариант А: Простые импланты (легко достать)
   - Вариант Б: Редкие импланты (сложно достать)
   - Вариант В: Легендарные импланты (очень редко)
   - Вариант Г: Разные уровни редкости
