## 2. Ограничения имплантов — ✅ Решение

### 2.1 Какие ограничения на импланты — ✅ Решение
- Слоты по типам (боевые, тактические, защитные, двигательные, OS)
- Совместимость (часть несовместима)
- Общий лимит количества имплантов
- Влияние класса, увеличение лимитов, визуальные проявления

### 2.2 Как работает лимит имплантов — ✅ Решение
- Базовый лимит для всех
- Бонусы от класса/прокачки
- Влияние человечности (снижение при потере), временное превышение лимита
- Влияние качества имплантов

### 2.3 Как работает энергетический лимит — ✅ Решение
- Общий энергетический пул + индивидуальные ограничения сложных имплантов
- Увеличение пула (прокачка/импланты)
- Влияние энергии на киберпсихоз (опционально), восстановление энергии
