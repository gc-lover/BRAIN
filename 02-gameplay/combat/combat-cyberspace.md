## 3. Киберпространство

**Киберпространство для хакерства:**
- Визуальное представление сетей, глубокое хакерство, защита
- Тренировочные бои, хабы, глобальные события, PvP/PvE, турниры
- Лор: после DataKrash доступ ограничен; требуется кибердека (имплант)

### 3.1 Тип киберпространства и его роль — ✅ Решение
- Полноценный режим + гибридный + модульная система (хабы/арены/события/глубина)

### 3.2 Структура киберпространства — ✅ Решение
- Многоуровневая + зональная: базовый взлом, хабы, арены, события, PvE-зоны, глубокие зоны
- Игроки/организации создают свои зоны

### 3.3 Тренировочные бои — ✅ Решение
- Отдельные тренировочные зоны, арены с наградами (без смертей), универсальные арены

### 3.4 Хабы — ✅ Решение
- Системные, пользовательские (приватные/публичные), корпоративные/организационные/клановые

### 3.5 Глобальные события — ✅ Решение
- Периодические + динамические + пользовательские события

### 3.6 PvP — ✅ Решение
- Безопасные зоны + PvP зоны + PvE зоны + PvPvE события

### 3.7 PvE контент — ✅ Решение
- Специальные зоны (рейды/инстансы) + интеграция в структуру сетей

### 3.8 Турниры и арены — ✅ Решение
- Системные + пользовательские + корпоративные турниры и арены

### 3.9 Доступ — ✅ Решение
- Базовый (всем с кибердекой), средний (от деки/навыков), продвинутый (Netrunner)
