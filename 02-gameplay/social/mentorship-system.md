## Концепция системы наставничества

### Важность системы наставничества

**Почему важна:**
- Teacher класс - один из ключевых классов в игре
- Наставничество создает глубокие социальные связи между игроками и NPC
- Обучение навыкам через наставничество - альтернативный путь прокачки
- Создание образовательного контента игроками для других игроков
- NPC могут обучать игроков различным навыкам

**Источники вдохновения:**
- **Path of Exile** - система обучения навыкам через мастера
- **EVE Online** - система обучения новых игроков ветеранами
- **KENSHI** - обучение через взаимодействие с NPC
- **Cyberpunk 2077** - различные учителя и наставники в мире игры

## Система наставничества

### 1. Типы наставничества

#### 1.1. Игрок (Teacher класс) → Другой игрок

**Концепция:** Игрок с классом Teacher может обучать других игроков

**Механика:**
- Игрок с классом Teacher может стать наставником для другого игрока
- Наставник обучает ученика навыкам и знаниям
- Ученик получает бонусы к прокачке навыков при обучении у наставника
- Наставник получает вознаграждение за обучение (опыт, репутация, ресурсы)

**Подклассы Teacher и их особенности:**

**Наставник (Mentor):**
- Фокус: Обучение новичков, помощь в освоении игры
- Особенности: Базовые навыки, обучение основам игры
- Бонусы: Максимальные бонусы к обучению новичков

**Профессор (Professor):**
- Фокус: Углубленное обучение, исследования, создание учебного контента
- Особенности: Продвинутые навыки, создание контента, исследования
- Бонусы: Создание учебных материалов, доступ к продвинутым ресурсам

**Тренер (Trainer):**
- Фокус: Обучение боевым/техническим навыкам, улучшение навыков других игроков
- Особенности: Боевые и технические навыки, физическая подготовка
- Бонусы: Максимальные бонусы к обучению боевым навыкам

**Условия для начала наставничества:**
- У наставника должен быть класс Teacher (любой подкласс)
- Ученик может быть любого класса
- Наставник должен иметь достаточный уровень навыка для обучения
- Ученик должен принять предложение наставничества

**Ограничения:**
- Один наставник может обучать ограниченное количество учеников одновременно
- Ученик может иметь только одного наставника одновременно (для конкретного навыка)
- Наставник должен иметь более высокий уровень навыка, чем ученик

#### 1.2. Игрок (Teacher класс) → NPC

**Концепция:** Игрок с классом Teacher может обучать NPC

**Механика:**
- Игрок с классом Teacher может обучать NPC различным навыкам
- Обучение NPC улучшает отношения с NPC
- Обученный NPC может помогать игроку в операциях
- Обученный NPC может получать уникальные возможности

**Особенности:**
- Обучение NPC требует много времени и ресурсов
- Обученный NPC может стать более эффективным помощником
- Обучение NPC может открыть доступ к специальному контенту

#### 1.3. NPC → Игрок

**Концепция:** NPC могут обучать игроков различным навыкам

**Механика:**
- NPC могут стать наставниками для игроков
- NPC обучают игроков навыкам в зависимости от своей специализации
- Обучение у NPC требует выполнения заданий и взаимодействия
- Обучение у NPC улучшает отношения с NPC

**Типы NPC-наставников:**
- **Боевые наставники** - обучение боевым навыкам (Solo, Lawman NPC)
- **Технические наставники** - обучение техническим навыкам (Netrunner, Techie NPC)
- **Социальные наставники** - обучение социальным навыкам (Fixer, Media NPC)
- **Классовые наставники** - обучение навыкам конкретного класса (NPC соответствующего класса)

**Условия для начала обучения у NPC:**
- Достаточная репутация с NPC
- Выполнение специальных квестов для доступа к обучению
- Доступность NPC для обучения (не все NPC могут обучать)
- Игрок должен соответствовать требованиям для обучения

**Особенности обучения у NPC:**
- Обучение у NPC обычно дешевле, чем у игроков
- NPC могут обучать только определенные навыки (в зависимости от специализации)
- Обучение у NPC может требовать выполнения специальных заданий
- Обучение у NPC может быть ограничено по времени или количеству учеников

#### 1.4. NPC → NPC

**Концепция:** NPC могут обучать друг друга (опционально, для живого мира)

**Механика:**
- NPC могут обучать друг друга навыкам
- Это делает мир более живым и динамичным
- Обучение NPC друг друга может влиять на игровой мир
- Это может быть фоновой механикой для создания динамики мира

## Механика обучения

### 2. Процесс обучения

#### 2.1. Начало обучения

**Шаги для начала обучения:**

1. **Поиск наставника/ученика**
   - Игрок ищет наставника через систему поиска
   - Игрок предлагает стать наставником другому игроку
   - Игрок обращается к NPC-наставнику

2. **Проверка условий**
   - Проверка уровня навыков наставника
   - Проверка репутации (для NPC)
   - Проверка доступности наставника

3. **Соглашение о наставничестве**
   - Наставник и ученик договариваются об условиях
   - Устанавливается вознаграждение (для обучения у игрока)
   - Определяется список навыков для обучения

4. **Начало обучения**
   - Создается связь наставник-ученик
   - Начинается процесс обучения

#### 2.2. Процесс обучения

**Типы обучения:**

**Теоретическое обучение:**
- Наставник объясняет теорию навыка
- Ученик получает знания и понимание
- Ускоряет прокачку навыка

**Практическое обучение:**
- Наставник показывает применение навыка
- Ученик практикует навык под руководством наставника
- Значительно ускоряет прокачку навыка

**Совместное выполнение заданий:**
- Наставник и ученик выполняют задания вместе
- Ученик получает опыт и прокачивает навыки
- Наставник получает вознаграждение за помощь

**Создание учебного контента:**
- Наставник создает учебные материалы (квесты, гайды, видео)
- Другие игроки могут использовать учебный контент для обучения
- Наставник получает вознаграждение за использование контента

#### 2.3. Прокачка навыков через наставничество

**Бонусы к прокачке:**
- **Ускорение прокачки:** Обучение у наставника ускоряет прокачку навыка (увеличивает получение опыта)
- **Качество обучения:** Высокий уровень навыка наставника увеличивает бонусы к прокачке
- **Специализация:** Наставник-специалист в навыке дает большие бонусы
- **Отношения:** Хорошие отношения между наставником и учеником увеличивают бонусы

**Формула бонусов (концепт):**
- Базовый бонус: +10-20% к опыту навыка
- Бонус от уровня наставника: +1-5% за каждые 10 уровней разницы
- Бонус от специализации: +5-15% если наставник-специалист
- Бонус от отношений: +1-10% в зависимости от репутации

#### 2.4. Завершение обучения

**Условия завершения обучения:**
- Ученик достиг желаемого уровня навыка
- Наставник или ученик решает прекратить обучение
- Автоматическое завершение при достижении максимума
- Расторжение договора по соглашению

**Результаты обучения:**
- Ученик получает прокаченные навыки
- Наставник получает вознаграждение (опыт, репутация, ресурсы)
- Улучшаются отношения между наставником и учеником
- Создается история обучения (для репутации)

## Уникальные способности Teacher класса

### 3. Активные способности

**Описание:** Временно ускоряет прокачку навыков для ученика

**Механика:**
- Наставник активирует способность для ученика
- Ученик получает временный бонус к прокачке навыка (+50-100% опыта)
- Длительность: 15-30 минут
- Кулдаун: 1-2 часа

**Особенности:**
- Работает только если наставник и ученик находятся вместе
- Можно использовать на нескольких учениках одновременно (зависит от уровня)
- Эффект увеличивается с уровнем наставника

## Влияние наставничества на отношения

### 5. Отношения наставник-ученик

#### 5.1. Развитие отношений

**Механика:**
- Наставничество улучшает отношения между наставником и учеником
- Длительное наставничество создает прочную связь
- Успешное обучение увеличивает репутацию наставника
- Плохое обучение или конфликты ухудшают отношения

**Уровни отношений:**
- **Начинающий** - только начали обучение
- **Активный ученик** - регулярное обучение
- **Верный ученик** - долгое и успешное обучение
- **Выпускник** - завершил обучение, но поддерживает связь

#### 5.2. Репутация наставника

**Механика:**
- Успешное обучение увеличивает репутацию наставника
- Репутация наставника влияет на доступ к ученикам
- Высокая репутация открывает доступ к премиум-ученикам
- Плохая репутация ограничивает доступ к ученикам

**Факторы репутации:**
- Количество успешно обученных учеников
- Качество обучения (уровень прокачки учеников)
- Отзывы учеников
- Создание качественного контента

## Особые механики

### 6. Специальные возможности наставничества

#### 6.1. Цепочка наставничества

**Концепция:** Ученик может стать наставником для других игроков

**Механика:**
- Выпускник может стать наставником для других игроков
- Цепочка наставничества создает иерархию обучения
- Наставник получает бонусы за обучение своих учеников
- Создается сеть обучения и знаний

#### 6.2. Групповое наставничество

**Концепция:** Несколько наставников могут обучать группу учеников

**Механика:**
- Группа наставников обучает группу учеников
- Разные наставники специализируются на разных навыках
- Групповое обучение более эффективно для больших групп
- Требует координации и организации

#### 6.3. Виртуальное обучение

**Концепция:** Обучение через виртуальную реальность или симуляцию

**Механика:**
- Наставник создает виртуальные сценарии для обучения
- Ученик практикует навыки в безопасной виртуальной среде
- Виртуальное обучение может быть более эффективным
- Требует специального оборудования или навыков

## Решения

### Система наставничества

**Решение (2025-11-03):** Полная система наставничества
- **Типы наставничества:** Игрок→Игрок, Игрок→NPC, NPC→Игрок, NPC→NPC (опционально)
- **Уникальные способности Teacher класса:** Ускоренное обучение, Создание контента, Мастер-класс

- **Влияние на отношения:** Развитие отношений, репутация наставника
- **Особые механики:** Цепочка наставничества, групповое обучение, виртуальное обучение

**Сложность системы:**
- Очень сложная система наставничества
- Множество механик и взаимодействий
- Глубокое влияние на прокачку и социальные отношения
