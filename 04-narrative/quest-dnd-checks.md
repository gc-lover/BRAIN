# Квесты и диалоги — Проверки в стиле D&D

**Статус:** draft  
**Версия:** 0.1.0  
**Дата создания:** 2025-11-05  
**Приоритет:** высокий

## Цель
Определить, где и какие проверки применяются в квестах, как они влияют на ветвления и какие требования к UI.

## Категории проверок
- Социальные: Убеждение (EMP), Обман (COOL), Запугивание (COOL), Харизма (EMP)
- Исследование: Восприятие (REF→Perception), Анализ (INT), Картография (INT)
- Тех/Хак: Взлом (TECH/INT), Ремонт (TECH), Крафт (TECH/INT)
- Действия: Акробатика/Паркур (AGI/REF), Скрытность (AGI/COOL), Внимание (REF)

## Схема проверки
- d20 + АтрибутныйМодификатор + НавыковыйБонус + СитуационныеМодификаторы >= DC
- Преимущество/Помеха: 2d20, лучший/худший.
- Криты: 1 — авто-провал, 20 — авто-успех, с особыми последствиями.

## Диапазоны DC для квестов (ориентиры)
- Социальные: 10/15/20/25 по уровню оппонента и контекста
- Тех/Хак: 12/17/22/27 по уровню устройства/сети
- Исследование: 10/15/20 с модификаторами окружения (-2..+2)

## Ветвления квестов (узлы и исходы)
Каждая точка ветвления описывается структурой:
- Узел: id, описание, тип проверки, DC, модификаторы
- Успех: последствия (новая ветка, награды, скрытая информация)
- Провал: последствия (альтернативный путь, штрафы, агро, тревога)
- Крит.успех: сильный бонус/скрытая ветка
- Крит.провал: усиливает риск/блокирует краткий путь

## UI-требования
- Показывать тип проверки и скрытую/пороговую сложность (опционально в настройках сложности игры)
- Индикация преимущества/помехи и источников модификаторов
- История бросков (лог сцены) для коопа и честности

## Врезки для основных квестов (встраивание проверок)
- 1.1 «Первые шаги»:
  - Обучающие проверки: Восприятие (10), Паркур (10), Общение (10)
- 1.2 «Выбор пути»: 
  - Социальные проверки для выбора фракции: Убеждение/Харизма (15–20)
- 1.3 «Развитие навыков»: 
  - Тех/Хак: Взлом (17–22), Ремонт (15–20)
- 2.1 «Выбор стороны»:
  - Соц-проверки высокого уровня (20–25), последствия на репутацию
- 4.2 «Взлом заслона»:
  - Слоистые проверки: серия Взломов (22→25→27) с нарастающим риском
