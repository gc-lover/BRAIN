## Узлы

### N-1 Туториал: Восприятие улицы
- Тип: Исследование / Perception
- Формула: d20 + REF мод + Perception бонус + мод.окружения ≥ DC 10
- Модификаторы: освещение +1; шум -1
- Успех: помечает 2 скрытных интерактивных объекта, +микро-XP
- Провал: ничего не помечает
- Крит.успех (20): +редкая подсказка о скрытом луте
- Крит.провал (1): временная помеха «перегрузка сенсоров» (-1 Perception на 2 мин)

### N-2 Туториал: Паркур через баррикаду
- Тип: Действие / Parkour
- Формула: d20 + AGI/REF мод + Parkour бонус ≥ DC 10
- Преимущество: активный учебный баф
- Успех: быстрый проход, +микро-XP
- Провал: задержка 3–5 сек
- Крит.успех: стилизованная анимация, +доп.микро-XP
- Крит.провал: падение, мелкий урон выносливости

### N-3 Социалка: Короткий диалог с фиксером
- Тип: Социальная / Communication
- Формула: d20 + EMP/WILL мод + Communication/Charisma бонус ≥ DC 10
- Успех: скидка у стартового торговца 2% на 1 ч.
- Провал: без скидки
- Крит.успех: скидка 5% + наводка (POI)
- Крит.провал: -1 отношение у нейтрального NPC на 24 ч.

### N-4 Стелс-микроэпизод (опционально)
- Тип: Скрытность / Stealth
- Формула: d20 + AGI/COOL мод + Stealth бонус + укрытие ≥ DC 10
- Преимущество: тень/укрытия
- Успех: пройдён участок без тревоги → альтернативный быстрый путь
- Провал: мелкий «агро» (NPC осматривается), без боя
- Крит.провал: вызов патруля (учебный), безопасный выход

### N-5 Тех-туториал: Вскрытие двери
- Тип: Хак/Ремонт (TECH/INT)
- Формула: d20 + TECH/INT мод + Hacking/Repair бонус ≥ DC 12
- Ситуация: таймер 30с, каждая повторная попытка -2
- Успех: открыта дверь, +микро-лут
- Провал: задержка, -1 расходуемый ресурс
- Крит.провал: учебная тревога (без штрафов к репутации)

## Групповые правила (MMORPG)
- N-2 и N-5: кооперативный бонус +1 за каждого ассистента (макс +3)
- Общий лог бросков для группы сохраняется в сцене
