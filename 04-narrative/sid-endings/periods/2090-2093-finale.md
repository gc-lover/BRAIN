# 2090-2093: ФИНАЛ ЛИГИ (Finale)

**Период:** 2090-2093 (финальные 3 года)  
**Фаза лиги:** Финал (100%)  
**Влияние:** Абсолютное — последний акт истории  
**Важность:** МАКСИМАЛЬНАЯ — определяет концовку лиги и параметры следующей

---

## Описание периода

**Контекст:** Последние 3 года лиги. Все ветвления предыдущих 70 лет сходятся к финалу. Игроки видят результаты ВСЕХ своих действий. Мир достигает критической точки. Симуляция готовится к перезапуску.

**Ключевые события:**
- **2090:** Финальные выборы, определение концовок
- **2091:** Развитие концовок, видимые изменения мира
- **2092:** Кульминация событий, судьбы фракций и NPC
- **2093:** ФИНАЛЬНЫЕ ТИТРЫ, перезапуск симуляции

**Уникальность:**
- **Видимая концовка:** Каждый день игроки видят приближение финала
- **Глобальные титры:** В конце все игроки видят единый финал
- **Персонализация:** Каждый игрок видит свой вклад
- **Эмоциональность:** Финал должен вызывать эмоции (как Cyberpunk 2077 × 1000)

## СИСТЕМА ОПРЕДЕЛЕНИЯ КОНЦОВОК

### Параметры влияния (собираются за всю лигу 2020-2093)

**1. Фракционный баланс:**
```
corporate_power: 0-100  (корпорации)
gang_power: 0-100       (банды)
independent_power: 0-100 (независимые)
```

**2. Региональное состояние:**
```
regions[region_name].prosperity: 0-100
regions[region_name].stability: 0-100
regions[region_name].tech_level: 0-100
regions[region_name].freedom: 0-100
```

**3. Глобальные оси:**
```
freedom_vs_control: -100 (тоталитаризм) ... 0 (баланс) ... +100 (анархия)
order_vs_chaos: -100 (хаос) ... 0 (баланс) ... +100 (порядок)
humanity_vs_transhumanism: -100 (машины) ... 0 (баланс) ... +100 (человечность)
tradition_vs_progress: -100 (традиции) ... 0 (баланс) ... +100 (прогресс)
```

**4. Технологические показатели:**
```
tech_level: 0-100 (общий уровень технологий)
cyberpsychosis_level: 0-100 (уровень киберпсихоза)
ai_development: 0-100 (развитие ИИ)
implant_adoption: 0-100 (использование имплантов)
```

**5. Симуляция:**
```
simulation_awareness: 0-100 (осознание симуляции)
simulation_acceptance: -100 (отрицание) ... 0 (игнор) ... +100 (принятие)
simulation_rebellion: 0-100 (восстание против системы)
```

**6. Состояние мира:**
```
world_stability: 0-100
world_prosperity: 0-100
world_violence: 0-100
world_inequality: 0-100
```
