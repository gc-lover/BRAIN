# 2030–2045 — Ветки по классам

## Конфигурация эпохи
- Купольные режимы (налоги/охрана/PvP), нейроэтика 1.0, контрабанда пропускной способности, дата‑убежища.

### Solo
- Аренные/охранные контракты куполов, приватные рейды
- D&D: Combat 18–22; Crit.success → статус «Преторианец Купола»

### Netrunner
- Чистка заражённых микрокодов, защита каналов/логирования
- D&D: Hacking 20–24; Crit.fail → «баг недели» (серверный дебафф)

### Techie
- Тех‑арбитраж нейроэтики, восстановление «Ковчегов»
- D&D: Tech/Repair 18–22; Advantage при доступе к мастерской

### Fixer
- Контракты нейроэтики/прошивок, серые прайсы
- D&D: Trading/Persuasion 18–22; Crit.success → эксклюзивные допуски

### Rockerboy / Media
- Анти‑редактура «Тихой Сводки», публичные слушания
- D&D: Charisma/Investigation 18–22; Crit.success → политическое окно

### Nomad
- Контрабанда пропускной способности, экстракция «пластов памяти»
- D&D: Driving/Stealth 18–22; Crit.fail → блокпост/изъятие груза

### Corpo
- Ликвидаторские тендеры: юридизация PvP‑диверсий
- D&D: Deception/Legal 18–22; Conseq: межкорп‑репутации

### Lawman
- Регламенты куполов/DAO, рейды на пиратские комбайнеры
- D&D: Intimidation/Investigation 18–22; Crit.success → расширенные полномочия

### Medtech
- Клинические пути нейроэтики, серые импланты
- D&D: Medicine 18–22; Crit.fail → смертность (соц‑удар)

### Politician
- «Устав Куполов», тарифы/режимы, этические шкалы
- D&D: Social/Legal 18–22; Conseq: глобальный сдвиг параметров района

### Trader
- Сетевые квоты/лицензии, серые рынки имплантов
- D&D: Trading 18–22; Crit.success → оптовые каналы

### Teacher
- Обучение нейроэтике/гигиене каналов
- D&D: Communication 16–20; Advantage в куполах с грантами
