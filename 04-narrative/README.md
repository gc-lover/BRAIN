# Повествование и квесты

Этот раздел содержит информацию о повествовании, системе квестов и диалогах в игре NECPGAME.

## Структура

- `quest-system.md` - Система квестов
- `branching-stories.md` - Ветвление сюжета
- `dialogues.md` - Диалоги
- `romances.md` - Романтические линии

## Источники вдохновения

- **BALDUR GATE 3** - система ветвления мира, выбор влияет на сюжет
- **The Witcher 3 Wild Hunt** - повествование, квесты, взаимодействие с персонажами
- **OBLIVION/SKYRIM** - RPG составляющая, фракции, сюжеты академий
